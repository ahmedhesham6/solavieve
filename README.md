
# Solavieve Task

Develop the feedback page following the design using React.


## Run Locally

Clone the project

```bash
  git clone https:
```

Go to the project directory

```bash
  cd solavieve
```

Frontend Installation

```bash
  cd frontend
  npm i
```

Backend Installation

```bash
  cd ../backend
  npm i
```
#### Migration
After setting the `DATABASE_URL` in the `.env` file
```bash
npm run migrate up
```

Start the server

```bash
  cd ..
  npm start
```
